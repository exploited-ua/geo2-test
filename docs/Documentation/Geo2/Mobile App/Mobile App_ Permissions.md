# Mobile App: Permissions

When you open the mobile app for the first time, it shows the permissions it requires.  Please grant all the permissions for Geo2 to function properly.  After the permissions are granted, the app automatically redirects you to the sign-in page.  Please also note that you can always allow permissions in the device settings both on Android and iOS.  The permissions are as follows:

|:----------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Location tracking     | For location tracking in routes, Proofs of deliveries (POD’s), vehicle checks, and public tracking.
